# Employee Management System

This is a Python and MySQL based Employee Management System developed as a CBSE Computer Science project.

## Features
- Secure Login System
- Employee Registration
- Salary Management
- Performance Tracking
- MySQL Database Connectivity

## Requirements
- Python 3.x
- MySQL Server
- mysql-connector-python

## How to Run
1. Install Python and MySQL
2. Install dependency:
   pip install mysql-connector-python
3. Run the program:
   python ashu.py
